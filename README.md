### 3.Funktioner
#### Contact Form Validation
Checks that email and message are filled in correctly
Displays error messages if the input is invalid
Limits the message length and shows a character counter
#### Responsive Design
Adapted for desktop, tablet (≤1024px), and mobile (≤600px)
Uses Flexbox and media queries for layout
Images and tables scale correctly on smaller screens

### 4.Git & versionering
All code is tracked with Git
Meaningful commits are used, for example:
Add contact form validation
Fix responsive layout on mobile
The README.md contains setup instructions so the project can be run directly
